# Realtime Tanks Demo — Construct 3

![Screenshot](screenshot.webp)

2D web client for [Realtime Tanks Demo](../README.md) built with [Construct 3](https://www.construct.net/) and the [Colyseus Construct 3 SDK](https://docs.colyseus.io/getting-started/construct3).

## Setup

1. Install the [Colyseus Construct 3 plugin](https://docs.colyseus.io/getting-started/construct3)
2. Open [Construct 3](https://editor.construct.net/) and use **Menu → Project → Open local project folder** to open this directory
   > **Note:** "Open local project folder" requires your browser to support the File System Access API. In Chrome/Edge, enable `chrome://flags/#file-system-access-local` if prompted.
3. Press **Preview** (or **F5**) to run

Make sure the [game server](../server/) is running on port 2567.

## Controls

- **WASD / Arrow keys** — Move
- **Mouse** — Aim turret
- **Left click** — Shoot
