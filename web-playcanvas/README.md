# PlayCanvas Client

3D web client built with PlayCanvas Engine, TypeScript, and Vite.

## Setup

```bash
npm install
```

## Running

Make sure the server is running first (`cd ../server && npm run dev`), then:

```bash
npm run dev
```

Open the URL shown in the terminal (default: http://localhost:5173).

## Building

```bash
npm run build
```

Output goes to `dist/`.
