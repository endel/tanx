# Realtime Tanks Demo — Defold

![Screenshot](screenshot.webp)

2D top-down client for [Realtime Tanks Demo](../README.md) built with [Defold](https://defold.com/) and the [Colyseus Defold SDK](https://docs.colyseus.io/getting-started/defold).

## Setup

1. Open the project in the [Defold Editor](https://defold.com/download/)
2. Select **Project → Fetch Libraries** to download dependencies
3. Press **Project → Build** to run

Make sure the [game server](../server/) is running on port 2567.

## Controls

- **WASD / Arrow keys** — Move
- **Mouse** — Aim turret
- **Left click** — Shoot
