# Tank Battle Multiplayer — GameMaker

![Screenshot](screenshot.webp)

2D client for [Tank Battle Multiplayer](../README.md) built with [GameMaker](https://gamemaker.io/) and the [Colyseus GameMaker SDK](https://docs.colyseus.io/getting-started/gamemaker).

## Setup

1. Open `TankBattle/TankBattle.yyp` in GameMaker Studio 2 (2024.x or later)
2. Download and import the [Colyseus GameMaker SDK](https://docs.colyseus.io/getting-started/gamemaker)
3. Press **F5** or click Run

Make sure the [game server](../server/) is running on port 2567.

## Controls

- **WASD / Arrow keys** — Move
- **Mouse** — Aim turret
- **Left click** — Shoot
